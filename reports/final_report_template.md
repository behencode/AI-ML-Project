# Intelligent Reading Comprehension and Quiz Generation System

## Abstract
This project implements a traditional-machine-learning reading comprehension and quiz generation system using the RACE dataset. The system includes a preprocessing pipeline, Model A (answer verification) trained with multiple classifiers and ensembles, Model B (distractor and hint generation), evaluation tooling with classification and generation metrics (BLEU, ROUGE-L, METEOR), and a Streamlit UI for interactive demonstration. Key results and visualizations are included in the Figures directory.
